## 🚀 Smart Dev - הרצה חכמה של סביבת פיתוח {#smart-dev}

### מה זה עושה?

סקריפט חכם שמטפל בכל מה שצריך להרצת סביבת פיתוח:

✅ בודק אם השרת כבר רץ (ומציע לעצור אותו)  
✅ בונה את הפרונטאנד **רק אם יש שינויים**  
✅ מריץ migrations  
✅ מפעיל את השרת  

### שימוש בסיסי

```powershell
.\smart-dev.ps1
```

### אפשרויות מתקדמות

```powershell
# אילוץ בנייה מחדש (גם אם אין שינויים)
.\smart-dev.ps1 -Force

# דילוג על בנייה (רק הרצת שרת)
.\smart-dev.ps1 -SkipBuild

# שינוי פורט
.\smart-dev.ps1 -Port 8002
```

### מה קורה מאחורי הקלעים?

1. **בדיקת שרת קיים** - אם השרת כבר רץ, מציע לעצור אותו
2. **בדיקת שינויים** - משווה תאריכי שינוי בין `frontend/src` ל-`frontend/dist`
3. **בנייה חכמה** - בונה רק אם יש שינויים או אם אין `dist`
4. **הרצת migrations** - מוודא שהמסד נתונים עדכני
5. **הפעלת שרת** - מפעיל את uvicorn עם כל ההגדרות הנכונות

### יתרונות

- ⚡ **מהיר** - לא בונה אם אין צורך
- 🧠 **חכם** - מזהה שינויים אוטומטית
- 🛡️ **בטוח** - בודק אם השרת כבר רץ
- 📊 **ברור** - מציג סטטוס ברור בכל שלב

```powershell
.\.venv311\Scripts\python.exe scripts/analyze_errors.py
```

## 📜 סקריפטים קיימים {#existing-scripts}

### `dev.ps1` - הרצה מהירה (פורט 8000)

```powershell
.\dev.ps1
```

- עוצר תהליכים על פורט 8000
- מפעיל שרת פיתוח מהיר
- **לא** בונה פרונטאנד
- מתאים לפיתוח backend בלבד

### `start.ps1` - הרצה מלאה (פורט 8001)

```powershell
.\start.ps1
```

- בונה פרונטאנד אם אין `dist`
- מריץ migrations
- מפעיל שרת מלא (API + Frontend)
- **תמיד** בונה אם אין `dist`

## 🎯 מתי להשתמש בכל סקריפט?

| סקריפט | מתי להשתמש | יתרונות | חסרונות |
|--------|-----------|---------|---------|
| `smart-dev.ps1` | **רוב הזמן** - פיתוח יומיומי | חכם, מהיר, בונה רק אם צריך | - |
| `check-status.ps1` | לפני תחילת עבודה, לאבחון בעיות | מציג תמונה מלאה | לא מפעיל כלום |
| `dev.ps1` | פיתוח backend בלבד | מהיר מאוד | אין פרונטאנד |
| `start.ps1` | הרצה ראשונה, אחרי git pull | מלא ומקיף | תמיד בונה אם אין dist |

## 💡 טיפים וטריקים

```powershell

.\check-status.ps1
```

```powershell

.\smart-dev.ps1 -Force
```

```powershell

.\smart-dev.ps1 -SkipBuild
```

## 🆘 פתרון בעיות נפוצות

### השרת לא עולה

```powershell
# 1. בדוק סטטוס
.\check-status.ps1

# 2. עצור תהליכים ישנים
Get-Process python | Stop-Process -Force

# 3. נסה שוב
.\smart-dev.ps1
```

### הפרונטאנד לא מתעדכן

```powershell
# אילוץ בנייה מחדש
.\smart-dev.ps1 -Force
```

### שגיאות בבנייה

```powershell
# התקן תלויות מחדש
cd frontend
npm install
cd ..

# בנה מחדש
.\smart-dev.ps1 -Force
```

### בעיות עם virtual environment

```powershell
# הפעל ידנית
.\.venv311\Scripts\Activate.ps1

# בדוק Python
python --version

# התקן תלויות
pip install -r requirements.txt
```
