המודל `Salesperson` לא מגביל שמות כפולים:
```python
class Salesperson(Base):

    id: Mapped[int] = mapped_column(primary_key=True)
    name: Mapped[str] = mapped_column(String(200), nullable=False)  # ללא unique=True
```
